## Honest notes
- The alias uses router transport (ccr) because Zen's free models use OpenAI-compatible
  format (/v1/chat/completions), not Anthropic native format. Claude Code talks to Zen
  through ccr, which translates the protocol.
- big-pickle is a stealth model — the actual model served may vary (observed as
  deepseek-v4-flash). This is by design per OpenCode's documentation.
- The in-process LLMsVerifier step reports (unverified) for the same pre-existing
  ~/api_keys.sh set -u issue affecting all providers. The direct HTTP round trip is the
  authoritative proof.
